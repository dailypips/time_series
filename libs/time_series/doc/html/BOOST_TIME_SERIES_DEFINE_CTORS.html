Macro BOOST_TIME_SERIES_DEFINE_CTORS

BOOST_TIME_SERIES_DEFINE_CTORS — For use by types derieved from time_series_facade<> to define their forwarding constructors.

Synopsis

BOOST_TIME_SERIES_DEFINE_CTORS(series_name)

Description

For use by types derieved from time_series_facade<> to define their forwarding constructors. The series_name argument is the name of the derived time series type. The macro defines the following constructors:

  • Default constructor

  • Copy constructor

  • Converting constructor from any TimeSeries model

  • Converting constructor from any ArgumentPack model

  • Constructor overloads that take up to BOOST_PARAMETER_MAX_ARITY named or positional arguments.

The series_name type must have an associated specialization of boost::constructors::construct<> that specifies the named parameters of the series_name constructors.
