Function shift

boost::time_series::shift — Shifts a TimeSeries by the specified offset.

Synopsis

template<typename Series, typename Offset> 
  shifted_series< Series > const shift(Series & series, Offset offset);
template<typename Series, typename Offset> 
  shifted_series< Series const  > const 
  shift(Series const & series, Offset offset);

Description

Shifts a TimeSeries by the specified offset. The offset can be positive, which shifts the series right, or negative which shifts it left.

Parameters:
offset

The offset by which the series is to be shifted.

series

The series to shift.

Requires:

Series is a model of the TimeSeries concept.

Offset is convertible to the offset_type of the input series.

Returns:

A shifted_series<> view that models TimeSeries.

Notes:

The complexity is O(1).
