Function make_ordered_inserter

boost::time_series::make_ordered_inserter — Creates a new ordered_inserter<Series> object.

Synopsis

template<typename Series, typename Offset> 
  ordered_inserter< Series > 
  make_ordered_inserter(Series & series, Offset offset);
template<typename Series> 
  ordered_inserter< Series > make_ordered_inserter(Series & series);

Description

Creates a new ordered_inserter<Series> object.

Parameters:
offset

The offset of the first run written into the resulting inserter, if the offset of that run is not otherwise specified.

series

The series into which this inserter writes.

Returns:

If offset is specified, returns ordered_inserter<Series>(series, offset). Otherwise, returns ordered_inserter<Series>(series)
