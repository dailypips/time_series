Function invert_elements

boost::time_series::invert_elements — Construct a new series from the old where each element is inverted.

Synopsis

template<typename Series, typename Out> 
  ordered_inserter< Out > 
  invert_elements(Series const & series, ordered_inserter< Out > out);
template<typename Series> Series const invert_elements(Series const & series);

Description

Construct a new series from the old where each element is inverted.

Parameters:
out

The inserter into which the new series is written.

series

The series to invert.

Requires:

Series is a model of the TimeSeries concept.

Out is a model of the OrderedInserter concept.
