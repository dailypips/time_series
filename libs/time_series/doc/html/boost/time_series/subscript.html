Function subscript

boost::time_series::subscript — Builds a new series from an existing series and an "index" series.

Synopsis

template<typename Series, typename Indices, typename Out> 
  ordered_inserter< Out > 
  subscript(Series const & series, Indices const & indices, 
            ordered_inserter< Out > out);
template<typename Series, typename Indices> 
  sparse_series< typename concepts::TimeSeries< Series const  >::value_type typename concepts::TimeSeries< Series const  >::discretization_type, typename concepts::TimeSeries< Indices const  >::offset_type > 
  subscript(Series const & series, Indices const & indices);

Description

Builds a new series from an existing series and an "index" series, where the new series has the following values and offsets:

  • (series[ index_offset_0 + index_value_0 ], index_offset_0)

  • (series[ index_offset_1 + index_value_1 ], index_offset_1)

  • (series[ index_offset_2 + index_value_2 ], index_offset_2)

  • ...

where index_offset_N is the offset of the N-th run in the index series, and index_value_N is the value of the N-th run in the index series.

Parameters:
indices

The index series.

out

The ordered inserter to receive the adjacent difference result.
series

The input series.

Requires:

Series is a model of the TimeSeries concept.

Indices is a model of the TimeSeries concept.

Out is a model of the OrderedInserter concept.

For variaibles x of the Indices value_type and y of the Indices offset_type, the expression (x+y) must be convertible to Series offset_type.

Returns:

A sparse_series<> if no ordered inserter is specified; otherwise, the ordered inserter.

Notes:

If using the version that takes an OrderedInserter, you must call .commit() on the returned OrderedInserter when you are done with it.
