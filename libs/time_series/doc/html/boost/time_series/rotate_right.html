Function rotate_right

boost::time_series::rotate_right — Rotates a series right, in the sense that the i-th run assumes the value of the (i-1)-th run. The first run in the series assumes the value of the value parameter if specified; otherwise, the first run is dropped.

Description

Rotates a series right, in the sense that the i-th run assumes the value of the (i-1)-th run. The first run in the series assumes the value of the value parameter if specified; otherwise, the first run is dropped.

Parameters:
out

The ordered inserter to receive the adjacent difference result.
series

The input series.

value

The value the first run in the series should assume.

Requires:

Series is a model of the TimeSeries concept.

Out is a model of the OrderedInserter concept.

Returns:

A sparse_series<> if no ordered inserter is specified; otherwise, the ordered inserter.

Notes:

If using the version that takes an OrderedInserter, you must call .commit() on the returned OrderedInserter when you are done with it.
