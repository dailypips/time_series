Function template operator==

boost::time_series::operator== — Compare two series for equality.

Synopsis

template<typename Left, typename Right> 
  bool operator==(time_series_base< Left > const & left, 
                  time_series_base< Right > const & right);

Description

Requires:

Left is a model of the TimeSeries concept.

Right is a model of the TimeSeries concept.

Returns:

True if the two series have the same discretization and the same values at the same offsets; false, otherwise.
