Acknowledgements

Many people contributed to the design and implementation of Boost.Time_series. I would like to thank Daniel Egloff of Zürcher Kantonalbank for helping to conceive the library and for his many suggested improvements. Matthias Troyer specified the library and contributed usage examples which gave the project its early direction. David Abrahams made innumerable suggestions and contributed the underlying code for the Sequence concept and the tag-dispatched customization points. Daniel Wallin wrote the python wrapper and several of the numeric algorithms.

I would like to thank Zürcher Kantonalbank for sponsoring the work on Boost.Time_series and graciously donating it to the community.
